    mkPath

    Mel Command

    tColor <string> -noOutliner -noViewport -noShape -noNode

    Description

    Set the color of the selected objects in maya.

    The color is defined by a string that can be a color name or a hexadecimal value. The hexadecimal value must be a string that starts with a '#' followed by 6 hexadecimal characters. Example : "#FF0000" for red.

    The color name can be one of the following :

    red
    pink
    purple
    deepPurple
    indigo
    blue
    lightBlue
    cyan
    teal
    green
    lightGreen
    lime
    yellow
    amber
    red_100
    pink_100
    purple_100
    deepPurple_100
    indigo_100
    blue_100
    lightBlue_100
    cyan_100
    teal_100
    green_100
    lightGreen_100
    lime_100
    yellow_100
    amber_100

    By default, the color is applied to the outliner, the viewport, the shape and the node of the object. You can specify which part of the object you want to color by using the flags : - -noOutliner : Do not color the outliner. - -noViewport : Do not color the viewport. - -noShape : Do not color the shape.

    Arguments

    • <string> : The color to set. Can be a color name or a hexadecimal value. Example : "red" or "#FF0000".

    • -noOutliner : Do not color the object in the outliner.

    • -noViewport : Do not color the object in the viewport.

    • -noShape : Do not color the shape of the object.

    Examples

    tColor "red"
// Set the color of the selected objects to red.

    tColor "#FF0000"
// Set the color of the selected objects to red.

    tColor "red" -noOutliner
// Set the color of the selected objects to red. Do not color the outliner.

    tColor "red" -noViewport
// Set the color of the selected objects to red. Do not color the viewport.
